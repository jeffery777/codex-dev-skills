# Excluded Packs

Version 1 intentionally includes only material derived from the `review-workflow-codex` and `review-workflow-codex-desktop` packs.

## `review-workflow`

Excluded because it is outside the first public Codex-only scope. The public repository focuses on Codex CLI and Codex Desktop software development workflows.

## `pptx-codex`

Excluded because presentation and slide workflows are not software development review or delivery workflows.

## `frontend-ui-codex-shared`

Excluded from version 1 because it was not sufficiently verified for this public baseline.

## Knowledge / Obsidian Capture Workflows

Knowledge, Obsidian, and vault capture workflows are excluded from version 1 because they are not necessary for the public software development review and delivery baseline.
